# React 项目总结

## 项目总结：
- 架构：路由，鉴权，页面部布局 目录结构
- 技术栈：umi, dva, redux-soga, react hooks 
- 通用工具： webpack esbuild, react-dom,react-router
- 代码检查和美化： eslint, prettier
- 数据持久化：immutable.js，immer.js
- 代码规范：Airbnb 

## react 常用插件
- 富文本编辑器：braft-editor
- 上传截图：antd-img-crop
- 吸顶 ： react-sticky
- 日期相关： dayjs、moment.js
- UI 库： antd，Material-UI，react-bootstrap
